FlexBox

Flexbox (Flexible Box Layout) in CSS is a layout model designed to help align and distribute space among items in a container.
Flexbox is one-dimensional, meaning it arranges items either in a row (horizontal) or a column (vertical), making it ideal for simpler, component-based layouts.
